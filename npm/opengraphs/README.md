# opengraphs-cli

Run OpenGraphs TUI with `npx`:

```bash
npx -y opengraphs-cli --help
npx -y opengraphs-cli run demo_train.py --auto autonomous
```

Or install globally:

```bash
npm install -g opengraphs-cli
opengraphs-cli --help
og --help
```

The package downloads the matching prebuilt `ogtui` release binary for your OS/arch and executes it.
